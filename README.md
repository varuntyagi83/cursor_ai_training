# CursorAI Masterclass | AI Catalyst (2026 Edition)

This repository contains a single page HTML training document for the **CursorAI Masterclass**, created as part of the **AI Catalyst Course (2026 Edition)**.

It is designed as a complete non technical guide that helps learners go from zero to AI powered developer, with a strong emphasis on practical workflows, structured learning modules, and a polished course website experience.

---

## What is included

- One full HTML course website
- Built in styling with modern dark theme UI
- Multi section structured learning content
- Navigation with smooth scroll and modular layout

Main file:

- `CursorAI-Masterclass-AI-Catalyst.html`

---

## How to run locally

### Option 1 (recommended)
Just open the HTML file in your browser.

1. Download or clone this repository
2. Open:

```bash
CursorAI-Masterclass-AI-Catalyst.html
```

### Option 2 (serve with a local web server)
If you prefer running it via localhost:

```bash
python3 -m http.server 8000
```

Then open:

```text
http://localhost:8000/CursorAI-Masterclass-AI-Catalyst.html
```

---

## Deploy on GitHub Pages (optional)

You can publish the training document as a live website using GitHub Pages.

1. Push this repository to GitHub
2. Go to **Settings** → **Pages**
3. Under **Build and deployment**
   - Source: Deploy from a branch
   - Branch: `main`
   - Folder: `/root`
4. Save

Your page will be available at a GitHub Pages URL after a short delay.
